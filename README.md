# grepline

Starting-point Bun monorepo scaffold with an Effect v4 CLI app in `apps/cli`.

## Layout

- `apps/cli/` for the hello-world Effect CLI package
- `scripts/` for repo scripts
- `.github/` for CI and shared actions
- `.changeset/` for release metadata
- `.specs/` for planning docs

## Package manager

This repo is set up for Bun:

```bash
bun install
bun run check
bun run dev
```
